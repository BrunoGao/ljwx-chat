## 🛠️ 工具脚本

### 核心工具

- **`rebuild-and-restart.sh`** ⭐ 一键重建和重启
- **`start-all.sh`** - 启动所有服务（MinIO + App）
- **`stop-all.sh`** - 停止所有服务

### 配置工具

- **`setup-minio.sh`** - 配置 MinIO（bucket + CORS）
- **`test-upload.sh`** - 诊断上传功能

### Docker 配置

- **`docker-compose.minio.yml`** - MinIO 服务
- **`docker-compose.lingjingwanxiang.yml`** - 应用服务
