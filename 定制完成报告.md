## 🚀 快速开始（2 种方式）

### 方式 1: 使用启动脚本（推荐）



### 可选：完整知识库功能

需要 PostgreSQL + PGVector：

```bash
# 1. 安装 PostgreSQL
brew install postgresql@14

# 2. 启动服务
brew services start postgresql@14

# 3. 创建数据库
createdb lingjingwanxiang

# 4. 安装 pgvector
psql lingjingwanxiang -c "CREATE EXTENSION vector;"

# 5. 更新 .env
echo "DATABASE_URL=postgresql://$(whoami)@localhost:5432/lingjingwanxiang" >> .env
```
