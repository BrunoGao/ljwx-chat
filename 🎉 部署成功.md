```sql

```

**向量相似度搜索示例:**

```sql
-- 需要先有嵌入数据
SELECT c.text, e.embeddings <-> '[0.1, 0.2, ...]'::vector AS distance
FROM chunks c
JOIN embeddings e ON e.chunk_id = c.id
ORDER BY distance
LIMIT 5;
```

```bash

```

**解决:**

```bash

```

### 问题：向量搜索不工作

**检查扩展:**

```bash
psql -d lingjingwanxiang -c "SELECT * FROM pg_available_extensions WHERE name = 'vector';"
```

**重新启用:**

```bash
psql -d lingjingwanxiang -c "CREATE EXTENSION IF NOT EXISTS vector;"
```
